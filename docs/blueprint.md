# **App Name**: ETF Weekly Rank

## Core Features:

- ETF Selection: Allow users to select ETFs from a predefined list.
- Weekly Rank Display: Display a ranked list of selected ETFs based on their weekly performance (price change).
- Performance Tagging: Automatically assign labels (e.g., 'Top Performer,' 'Average,' 'Underperformer') to each ETF based on its ranking. An LLM will be used as a tool for tagging the ETFs, providing appropriate disclaimers for each ETF, ensuring balanced outputs that reflect the intention of transparency described in the PRD.
- Visual Search: Enable Google Image Search integration on ETF names/icons for quick visual understanding.
- Disclaimer Display: Show data source and disclaimer at the bottom of the app.

## Style Guidelines:

- Primary color: Soft blue (#72BCD4), conveying trust and stability for financial information.
- Background color: Very light blue (#F0F8FF), providing a clean and calming backdrop.
- Accent color: A contrasting orange (#FF9800) to highlight top-performing ETFs and important actions.
- Headline font: 'Poppins', sans-serif, for titles and key information, offering a modern and accessible feel.
- Body font: 'PT Sans', sans-serif, for readability of ETF details and descriptions.
- Simple, neutral icons to represent ETFs and data points, avoiding complex financial imagery.
- Clean, single-page layout to display the ETF list prominently, emphasizing ranking and status labels.
